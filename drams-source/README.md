# DRAMS — Disaster Resource Allocation & Management System

## Setup

```bash
npm install
npm start
```

## Pages
- **Login** — Role-based auth (Admin / NGO / Volunteer)
- **Dashboard** — Admin overview with stats & activity log
- **Resource Inventory** — Asset tracking table with filters
- **Report Disaster** — Incident reporting form
- **Resource Allocation** — Deploy supplies to zones

## Stack
- React 18
- CSS Variables design system
- Google Fonts: Rajdhani + DM Sans + DM Mono
